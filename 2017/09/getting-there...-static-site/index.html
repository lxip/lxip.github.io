Getting there... static site

September 24, 2017 in frontend

I have been thinking for a while putting up a blog recording random stuff. It might be a better way to keep track what I’ve been looking at and regularize some massy notes. After wasting a summer leaving the plain template lying in my disk folder, it’s good to start now since the season is getting closer to my background picture.

With the free domain provided by Github, there has been a good amount of choices for set up a static website/blog. Here is the breif histroy of options I’ve gone through:

JekyII Hexo Hugo
Language Ruby NodeJs Go
Speed Moderate Fast VeryFast
Complexity Structural Structural Structural

Themes Good VeryGood Moderate

Hugo is using Go, a newer language in the rise, as powering source,gti commit -m it gives great speed and compatibility for the site and having a growing number of themes for blogs which makes it a good start.
